一个设计图都是有尺寸的，你要确定一下，常见的有640  750

适配：
    rem适配  思想：在大屏上面显示大盒子    在小屏上面显示小盒子    等比缩放
    百分比适配    在不同的屏上  所占的比例一样  
    flex适配  容器   项目   12个属性

今天要写的页面采用：
    没有采用rem适配   直接使用百分比+px就OK了
